Esteganografia

Escondendo informações em imagens

Isac Fadoni Leite

Renan Luiz Vendramini

O que é

Existem várias técnicas para esconder informações dentro de arquivos. Uma das mais utilizadas é distribuir a informação secreta dentro do ruído de imagens, músicas e vídeos. Esse ruído - que pode ser considerado a parte não útil do arquivo - é resultado de conversões e gravações, um processo natural nos arquivos atuais.

A mensagem também pode ser dividida e espalhada em vários arquivos, assim ninguém consegue descobrir todo o conteúdo sem juntar as partes diferentes. As informações são espalhadas aos poucos, dentro dos pixels de uma imagem, por exemplo. A mistura das variadas técnicas de esteganografia é o que torna ainda mais complicado desmantelar as informações escondidas, o que muitas vezes só é possível com os algoritmos exatos utilizados para a tarefa.

Passo a Passo

Para usar os benefícios da esteganografia, é preciso utilizar softwares específicos para essas camuflagens, abaixo, dois softwares e sua utilização.

Camouflage

Este é um dos aplicativos mais intuitivos em esteganografia básica.

  1. Baixar e Instalar: Para baixar vá para camouflage.unfiction.com e clique no botão ‘Download’, e instale-o normalmente.
  2. Clique sobre o arquivo que deseja esconder com o botão direito do mouse e selecione a opção ‘Camouflage’.
  3. Avance na próxima tela e em “Camouflage Using” é a hora de escolher o arquivo onde os dados serão escondidos. Você pode usar uma foto qualquer ou até mesmo outro documento nessa etapa.
  4. As últimas opções servem para escolher o lugar onde a imagem será salva e até mesmo aplicar uma senha para abri-la em outro local.
  5. Para abrir os dados camuflados repita o processo no novo arquivo, mas utilize a opção “Uncamouflage”.

Hide and Reveal
